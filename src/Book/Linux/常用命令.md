## 文件&目录

```bash
pwd: 			# 显示当前路径
ls: 			# 列出当前目录所有文件
ll: 			# 类似于ls，列出文件的同时，显示文件详细信息

touch: 			# 新建文件
mkdir: 			# 新建目录

rm: 		    # 删除文件
rm -r: 			# 删除文件夹
mv： 		   # 移动文件夹（）

reset: 			# 重新初始化终端
clear: 			# 清屏

histroy: 		# 查看命令历史
#： 			   # 注释

tar cvzf output.tar.gz file1 file2 file3....	# 打压缩包

sz file # 下载文件
rz # 上传文件

```
