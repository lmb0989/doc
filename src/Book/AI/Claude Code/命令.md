- ### 项目初始化

  使用 /init 命令 

  在新项目中，使用 `/init` 让 Claude Code 了解项目并创建 CLAUDE.md 配置文件：

  ```
  > /init
  ```

  Claude Code 会：

  1. 分析项目结构
  2. 识别使用的技术栈
  3. 创建 CLAUDE.md 文件记录项目约定

- ### 开始新对话 

  使用 `/clear` 命令清除当前对话历史，开始全新对话：

  ```
  > /clear
  对话已清除
  ```

- ### 继续上次会话 

  ```bash
  claude -c
  claude --continue
  ```

- ### 规划模式

  ```bash
  > /plan 重构用户认证模块，使用 JWT 替换 session
  ```

- ### 跳过权限确认

  ```bash
  claude --dangerously-skip-permissions
  ```

- ### 中断执行

  如果 Claude Code 正在执行某个操作，想要中断：

  - 按 `Escape` 键可以中断当前操作
  - 按 `Ctrl+C` 可以强制停止

- ### 其他命令

  ```bash
  /config	查看或修改配置
  /cost	查看当前会话的 Token 使用量
  /doctor	检查系统环境和配置
  /commit 智能生成提交信息并提交代码
  ```
